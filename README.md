# TeleEasy

**Short Description:**  
TeleEasy is a lightweight Minecraft mod that allows players to teleport instantly to the location they are looking at, saving time and hunger while enhancing gameplay quality.

---

## Features

• Instant teleportation to the block you are looking at  
• Lightweight and optimized for performance  
• Helps reduce hunger usage during long-distance travel  
• Fully configurable keybindings for convenience  
• Ideal for exploration, adventure, and long-distance travel

---

## Controls

By default, TeleEasy uses a single key to activate teleportation. Players can freely change the assigned key in Minecraft's control settings.

## Installation

1. Place the `TeleEasy-1.0.0+mc1.21.11.jar` file into your Minecraft `mods` folder.  
2. Make sure you are using **Fabric Loader 1.21.11** or higher.  
3. Launch Minecraft and enjoy instant teleportation!
